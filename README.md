# Auto Scroller

A browser extension to scroll the screen for you while you are doing something else. In my case, eating while reading an essay.

## Features
- **Automated Movement:** Scroll smoothly without using your mouse or touchpad.
- **Custom Intervals:** Precise control over the time (in seconds) between each scroll.
- **Adjustable Distance:** Set exactly how much of the screen height to move (set within the source code).
- **Toggle Control:** Easily start and stop the scroller via the extension popup.

## How to run locally
1. *Clone the repo:*
```
git clone git@github.com:navjotmaan/auto-scroller.git
cd auto-scroller
```
2. *Load into Chrome:*
- Open Chrome and navigate to [chrome://extensions/](chrome://extensions/).
- Enable Developer mode using the toggle in the top-right corner.
- Click the Load unpacked button.
- Select the auto-scroller folder from your local machine.

3. *Usage:*
- Pin the extension to your toolbar.
- Navigate to any website, open the popup, and set the interval!
